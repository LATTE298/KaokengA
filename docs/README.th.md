# Kaokeng — Daily Life (ชีวิตประจำวัน)

_ตรวจสอบล่าสุด: 2026-04-29_

แอปพลิเคชันกระบะทราย (sandbox) ที่ออกแบบมาเพื่อรองรับความหลากหลายทางระบบประสาท (neuro-inclusive) สำหรับเด็กไทย สร้างขึ้นด้วย Flutter และเอนจินเกม Flame ออกแบบมาเพื่อรองรับการใช้งานบนแท็บเล็ต Android ในแนวนอนเป็นหลัก

## ฟีเจอร์หลัก

- **โมดูล A — ชีวิตประจำวัน (Daily Life):** รูปแบบการเล่นเกมจำลองสถานการณ์แบบลากแล้ววาง (เช่น การซื้อนมที่ 7-Eleven) พร้อมเสียงอ่าน (TTS ภาษาไทย) แสดงความยินดี/ให้กำลังใจ
- **โมดูล B — เกมความจำ (Memory Game):** เกมการ์ดจับคู่ภาพ พร้อมคำศัพท์หมวดสัตว์ภาษาไทย
- **โมดูล C — กระดานเสียง (Sound Board):** รวมคำศัพท์ 30 คำจาก 5 หมวดหมู่ พร้อมระบบอ่านออกเสียง (TTS)
- **แดชบอร์ดสำหรับผู้ปกครอง (Parent Dashboard):** ระบบสมัครสมาชิกและเข้าสู่ระบบด้วยอีเมล/รหัสผ่าน, บันทึกประวัติการใช้งาน (activity log), และระบบเปิด/ปิดการเข้าถึงในแต่ละสถานการณ์จำลอง
- **เซสชันการใช้งานสำหรับเด็กแบบไม่ระบุตัวตน (Anonymous child sessions)** ผ่าน Firebase Auth — ไม่จำเป็นต้องสร้างบัญชีสำหรับเด็ก
- **ขับเคลื่อนเนื้อหาด้วย JSON:** สามารถเพิ่มสถานการณ์ใหม่ๆ ได้โดยไม่ต้องแก้ไขโค้ด

## เทคโนโลยีที่ใช้ (Tech Stack)

| เลเยอร์ (Layer) | ไลบรารี (Library) |
|---|---|
| UI เฟรมเวิร์ก | Flutter |
| เอนจินเกม | Flame 1.22 |
| การจัดการสเตต (State management) | Riverpod 2 |
| โมเดลข้อมูล | Freezed + json_serializable |
| การจัดการเส้นทาง (Routing) | go_router |
| แบ็กเอนด์ | Firebase Auth + Cloud Firestore |
| ระบบเสียง | just_audio |
| ฟอนต์ | Google Fonts |
| ไอคอน | Phosphor Flutter |

## เริ่มต้นการใช้งาน (Quick Start)

โคลนโปรเจกต์:

```bash
git clone https://github.com/Kenjeaw/Kaokeng.git
cd Kaokeng
```

ติดตั้ง dependencies:

```bash
flutter pub get
```

รันแอป:

```bash
flutter run
```

> สามารถดูข้อกำหนดเบื้องต้นแบบเต็ม, การตั้งค่า Firebase และคำสั่งสำหรับบิลด์แอปพลิเคชันได้ที่ [BUILD.md](BUILD.md)

## โครงสร้างโปรเจกต์ (Project Structure)

```text
lib/
├── main.dart                  # จุดเริ่มต้นของแอปพลิเคชัน (App entry point)
├── firebase_options.dart      # ไฟล์ตั้งค่า Firebase SDK (สร้างขึ้นอัตโนมัติ)
├── features/                  # คอนโทรลเลอร์ระดับฟีเจอร์
├── game/                      # คอมโพเนนต์ของเกม (Flame)
├── l10n/                      # ข้อความสำหรับระบบอ่านออกเสียง (TTS) ภาษาไทย
├── models/                    # โมเดลข้อมูล Freezed
├── providers/                 # Riverpod โพรไวเดอร์
├── routes/                    # ไฟล์ตั้งค่า go_router
├── screens/
│   ├── child/                 # หน้าจอสำหรับเด็ก
│   └── parent/                # หน้าจอแดชบอร์ดสำหรับผู้ปกครอง
├── services/                  # ลอจิกธุรกิจ และ Repositories
├── theme/                     # โทเค็นการออกแบบ (สี, รูปแบบตัวอักษร, ระยะห่าง)
└── widgets/                   # UI คอมโพเนนต์ที่ใช้ร่วมกัน

assets/
├── scenarios/                 # ไฟล์กำหนดสถานการณ์ (JSON)
├── images/                    # รูปภาพฉากและสิ่งของ
├── memory_packs/              # ชุดการ์ดสำหรับเกมความจำ
└── vocabulary/                # คำศัพท์สำหรับกระดานเสียง

specs/                         # เอกสารข้อกำหนดของผลิตภัณฑ์และการออกแบบ
test/                          # ไฟล์ทดสอบ Unit, Widget และ Game
```
